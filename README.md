# 北京市空气质量AQI可视乎

### 项目简介
本项目针对北京市空气质量指数(AQI) 的综合性可视化分析项目。基于2022年至2025年全市35个监测站点的逐时数据，通过多维度、多尺度的可视化技术，系统揭示北京市空气质量的时空分布规律与演变特征。

### 项目概览
- 数据来源: 王晓磊从中国环境监测总站获取的北京市空气质量监测数据 https://quotsoft.net/air/#archive
- 时间范围: 2022年1月1日 - 2025年11月5日
- 空间范围: 北京市16个市辖区，35个监测站点
- 分析维度: 时间趋势、空间分布、污染等级、区域对比
- 技术特点: 多尺度分析、标准可视化编码
